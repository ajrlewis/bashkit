## Installation

Install via pip:

```bash
pip install git+https://github.com/ajrlewis/bashkit.git
```

## Usage Examples

### Bitcoin Full Node Steps

Run the following scripts:

    1. bashkit/git/install.sh

    3. bashkit/bitcoin/install.sh
    4. bashkit/bitcoin/create_config.sh
